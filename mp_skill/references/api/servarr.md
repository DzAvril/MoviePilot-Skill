## servarr

- `DELETE /api/v3/movie/{mid}` — 删除电影订阅
- `DELETE /api/v3/series/{tid}` — 删除剧集订阅
- `GET /api/v3/languageprofile` — 语言
- `GET /api/v3/movie` — 所有订阅电影
- `GET /api/v3/movie/lookup` — 查询电影
- `GET /api/v3/movie/{mid}` — 电影订阅详情
- `GET /api/v3/qualityProfile` — 质量配置
- `GET /api/v3/rootfolder` — 根目录
- `GET /api/v3/series` — 所有剧集
- `GET /api/v3/series/lookup` — 查询剧集
- `GET /api/v3/series/{tid}` — 剧集详情
- `GET /api/v3/system/status` — 系统状态
- `GET /api/v3/tag` — 标签
- `POST /api/v3/movie` — 新增电影订阅
- `POST /api/v3/series` — 新增剧集订阅
- `PUT /api/v3/series` — 更新剧集订阅
